What does Jailbreak do?

Well, it does a lot of awesome things! But basically, it's the process by which full execute and write access is obtained on all the partitions of iOS, iPadOS, tvOS and watchOS. It's usually done by patching /private/etc/fstab to mount the System partition as 'read-write'.
